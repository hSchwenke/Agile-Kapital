# Controle Financeiro Pessoal

Um dashboard financeiro moderno, rápido e responsivo construído para ajudar no controle de receitas e despesas diárias. Desenvolvido com foco na experiência do usuário e interface limpa.

# Funcionalidades

- **Dashboard Dinâmico:** Visualização clara de Renda Fixa, Receitas Extras e Despesas.
- **Cálculo Automático:** O Saldo Atual é calculado em tempo real a cada nova transação.
- **Alertas Visuais:** O cartão de saldo muda de cor dinamicamente (verde para seguro, vermelho para alerta) quando o saldo atinge R$ 100 ou menos.
- **Design Responsivo:** Layout adaptável que funciona perfeitamente tanto no desktop quanto no celular (Media Queries).
- **Persistência de Dados:** As transações são salvas automaticamente no `LocalStorage` do navegador, para não perder os dados ao recarregar a página.
- **Extrato Inteligente:** Lista de transações com scroll customizado e prevenção de quebra de layout.

# Tecnologias Utilizadas

- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- CSS3 (Flexbox, Media Queries, Variáveis)
